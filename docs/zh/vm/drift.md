# Drift

> v 6.9.0 新增

## 概述

Drift 数据库查看器，支持对数据库进行可视化操作。

## 功能

- 查看数据库表
- 排序和过滤
- 导出 CSV
- 在编辑器中预览数据表
- 动态调整列宽
- 编辑列数据
- 删除表数据
- 动态执行数据表


## 使用方法

项目中需要添加 drift 相关依赖,并需要执行打开某个数据库相关的代码才会在插件中显示
