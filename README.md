# FlutterX 插件文档

这是 FlutterX 文档站的 MkDocs 源码仓库。

## 在线访问

- 主站: [https://flutterx.itbug.shop/](https://flutterx.itbug.shop/)
- GitHub Pages 镜像: [https://mdddj.github.io/flutterx-doc/](https://mdddj.github.io/flutterx-doc/)

## 本地开发

安装依赖：

```bash
python -m pip install -r requirements.txt
```

启动开发服务器：

```bash
mkdocs serve
```

构建静态站点：

```bash
mkdocs build --clean --strict
```

构建产物默认输出到 `site/`。
